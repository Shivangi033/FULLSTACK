# Multi-Page React App with Navbar Navigation

## Overview

This is a React single-page application (SPA) that implements a multi-page experience using React Router with a navigation bar. The app features a Profile page and a Dashboard page with a persistent navbar for easy navigation.

## Installation

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`
